# brainfuck

## Usage

`brainfuck.py [-h] brainfuck_file`

```
positional arguments:
  brainfuck_file  brainfuck (.bf or .b) file to execute

optional arguments:
  -h, --help      show this help message and exit
```

## Example
```
$ python brainfuck.py test/hello.bf
Hello, World!
```

```
$ python brainfuck.py test/mandelbrot.b > test/mandelbrot.txt
$ cat test/mandelbrot.txt
```
